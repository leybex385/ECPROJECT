# EasyCrypto SPA

A modern cryptocurrency single-page application (SPA) with a dark purple gradient fintech aesthetic.

## Features
- Dynamic coin lists (Hot, Gainers, Losers)
- Live price updates (Simulated)
- Exchange and Contract trading interfaces
- Premium SVG icons with glow effects
- Fast navigation and responsive design

## Technologies Used
- HTML5 / CSS3 (Vanilla)
- Vanilla JavaScript
- Google Fonts (Inter, Orbitron, Space Grotesk)
- SVG Vector Icons

## How to Run
Simply open `mobile.html` in any modern web browser.
